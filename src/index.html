Take a look at what you'll learn:

The JavaScript Marathon curriculum is a comprehensive program designed to equip aspiring frontend developers with the skills and knowledge needed to build robust, user-friendly web applications.

  • In this module, students will learn the basics of JavaScript, including data types, control structures, functions, objects, and arrays. They will also learn how to use JavaScript to manipulate the Document Object Model (DOM) and create dynamic web pages

  • This module covers the fundamentals of HTML and CSS, including document structure, layout, and styling. Students will learn how to use CSS to create responsive designs that adapt to different screen sizes and devices.

  • This module focuses on building web applications using the React framework. Students will learn how to use React to create reusable components, manage state, and handle user input.

  • This module covers the basics of Node.js and the Express framework for building server-side applications. Students will learn how to use Node.js to build RESTful APIs and handle server-side rendering.

  • In this module, students will learn how to test and debug their web applications using tools like Jest and Chrome DevTools.

  • This module focuses on project management skills, including project planning, task estimation, and version control using Git.
